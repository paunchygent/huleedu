# .claude/archive/

Historical and completed artifacts.

## Structure

- **code-reviews/** - Historical PR reviews and code analysis
  - PR review documents
  - Contract analysis
  - Testing infrastructure reviews

- **session-results/** - Completed session results and summaries
  - Documentation session results
  - README standardization results
  - Other session outputs

- **prompts/** - Historical prompt templates
  - Old continuation prompts
  - Phase-specific prompts
  - Migration prompts

- **repomix/** - Old repomix packages
  - Historical AI architect review packages
  - Context packages for specific tasks

- **task-archive/** - Archived task recommendations and completed work
  - Task archive recommendations
  - Old task documents
  - Completed implementation plans

- **reports/** - Completed research and investigation reports
  - Implementation plans extracted into tasks
  - Domain model investigations
  - API audit reports
  - Naming: `<subject>-YYYY-MM-DD.md`

## Purpose

This directory preserves historical context while keeping active work directories clean. Items here are kept for reference but are not part of current active work.

## Maintenance

Periodically review and:
- Remove very old artifacts (>6 months) if no longer relevant
- Compress large files
- Update this README with any new archive categories
