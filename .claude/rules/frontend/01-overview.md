# Frontend Overview

## Stack
- **Framework**: Vue 3 with Composition API
- **State**: Pinia
- **Routing**: vue-router
- **Styling**: TailwindCSS 4
- **Build**: Vite, pnpm
- **Types**: TypeScript, vue-tsc
- **Testing**: Vitest

## Development Root
Work from `/frontend/` directory as your development root.

## Commands
```bash
pdm run fe-prototype-build  # Build CSS
pdm run fe-dev              # Dev server with hot reload
```
