# Design System

## Color Palette
- **Canvas**: `#F7F5F0` (background)
- **Navy**: `#1C2E4A` (primary)
- **Burgundy**: `#7B2D3B` (accent)
- **Emerald**: Success states
- **Amber**: Processing states

## Text Opacity Hierarchy

| Opacity | Use |
|---------|-----|
| /100 | Primary text |
| /70 | Secondary text |
| /60 | Metadata (minimum readable) |
| /50 | Icons only |
